# Heartbeat v3 (Client)

Local-first, end-to-end-encrypted P2P chat client. Companion to [heartbeat-server](https://github.com/sahitkogs/heartbeat-server).

**Phase 10.1 milestone (this commit):** Ed25519 keypair generated on first launch; QR-code identity exchange between phones; local contacts list.

**Status:** in development. Not for general use yet.
