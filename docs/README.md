# heart-beat-v3 docs

heart-beat-v3 is the original **cloud-relay** client (Firebase/FCM, single hardcoded
relay). Active development has moved to **`heart-beat-local-android-app`** — the
cloud-free, multi-network, self-hosted fork. For the forward-looking roadmap (including
the features once planned here — voice memos, images, calls, multi-device), see that
repo's `docs/ROADMAP.md`.

**Current / reference docs kept at top level:**
- `privacy.md` — privacy policy (Play requirement)
- `10.2-crypto-choice.md` — crypto design decision (libsignal)
- `add/index.html` — contact-share web landing page

**`archive/`** — historical planning, design, test-result, and prompt docs from v3's
phased development. Retained for reference; not active.
